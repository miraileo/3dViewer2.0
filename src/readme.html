Инструкция по использованию 3DViewer v2.0

Установка:

  1. Вызвать цель make install из папки src.
  2. Дождаться установки.

Сборка в архив:

  1. Вызвать цель make dist из папки src.
  2. Архив собирается в папку src.

Удаление:

  1. Вызвать цель make uninstall из папки src.
  2. Дождаться удаления

Запуск и использование:

  1. Для запуска приложения написать make build или просто make
  2. Программа загружает файлы в формате *.obj.
  3. После загрузки объектного файла доступен весь функционал в графическом интерфейсе, перемещение по осям возможно при помощи соответствующих слайдеров на цифровой панели программы.
  4. Для выбора необходимого цветового решения необходимо использовать кнопки "PointColor", "EdgeColor", "Background color". Для сохранения в формате bmp, jpg использовать кнопку screenshot.
  • 3DViewer v1.0 поддерживает следующие действия:
    • Перемещение и вращение:
      1. Перемещение по оси X: "Translate X: от -180 до +180"
      2. Перемещение по оси Y: "Translate Y: от -180 до +180"
      3. Перемещение по оси Z:"Translate Z: от -180 до +180"
      4. Вращение по оси X: "Rotate X: от -180 до +180"
      5. Вращение по оси Y: "Rotate Y: от -180 до +180"
      6. Вращение по оси Z: "Rotate Z: от -180 до +180"
      7. Приближение: "Scale от 1 - до 100"
    • Опции:
      1. Тип проекции: "Central" - центральная, "ortho" - параллельная
      2. Тип вершин: "None" - невидимые, "Circle" - шарообразные , "Square" - квадратные
      3. Тип граней: "Solid - прямые, "Dashed" - пунктирные
      4. Толщина граней: "Справо от типа граней"
      5. Цвет вершин: "PointColor"
      6. Цвет граней: "EdgeColor"
      7. Цвет заднего фона: "Background color"
      8. Сохранение значений в формате: bmp или jpg "screenshot"
